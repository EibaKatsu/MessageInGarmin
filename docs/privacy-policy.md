# Privacy Policy for Zone coach

Last updated: 2026-03-02

`Zone coach` is a Garmin Connect IQ Data Field.

## Data Handling
1. The app does not collect personal data on external servers.
2. The app does not transmit data over the internet.
3. The app does not use third-party analytics or advertising SDKs.

## Device Data Used During Activity
The app reads activity data available from the watch during runtime:
1. Heart rate
2. Distance
3. Altitude
4. Timer information

This data is used only to generate on-device coaching messages in real time.

## Data Sharing
1. No user data is sold.
2. No user data is shared with third parties.
